## 为什么需要 JSON 在线解析工具？

这时候，一个在线工具比本地 IDE 更快更方便 —— 打开浏览器就能用，无需安装。

## JSON 格式优化的最佳实践

在使用 JSON 工具时，有几个技巧值得注意：

1. **先校验再格式化**：确保语法正确后再美化，避免格式化后的结果误导判断
2. **注意编码**：中文字符如果显示为 `\uXXXX`，说明被 Unicode 转义了，通常是正常的
3. **大文件先压缩**：超过 1MB 的 JSON 建议先压缩传输，到客户端再格式化
4. **善用节点折叠**：复杂 JSON 结构层级很深时，折叠不关心的节点能大幅提升可读性

## JSON 常见语法错误

| 错误 | 示例 | 正确写法 |
|------|------|----------|
| 键名没引号 | `{name: "test"}` | `{"name": "test"}` |
| 使用单引号 | `{'name': 'test'}` | `{"name": "test"}` |
| 尾部逗号 | `{"a":1, "b":2,}` | `{"a":1, "b":2}` |
| 有注释 | `{"a":1 // comment}` | `{"a":1}` |
