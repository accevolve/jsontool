JSON 在线校验工具

JsonTool.cn 提供专业的 JSON 校验服务，基于 JSONLint 标准对 JSON 数据进行严格的语法检查。将 JSON 粘贴到输入框，工具会实时校验并标注「✓ 校验通过」或「✗ 校验失败」及具体错误信息。

JSON 校验规则

本工具依据 RFC 8259（JSON 数据交换格式标准）进行校验，主要检查以下规则：

  • 所有键名必须使用双引号包裹
  • 字符串值必须使用双引号（不接受单引号）
  • 不允许尾部逗号（trailing comma）
  • 不支持注释（// 或 /* */）
  • 数值不允许前导零（如 01、00.5）
  • 特殊字符必须使用反斜杠转义
  • 大括号和方括号必须正确配对

JSON 校验失败如何修复？

  1. 查看错误提示中的行号和字符位置
  2. 检查该位置附近是否存在缺少引号、多余逗号等问题
  3. 修正后重新粘贴校验，直到显示「✓ 校验通过」
  4. 如果 JSON 来自程序生成，检查序列化代码是否使用了标准库
